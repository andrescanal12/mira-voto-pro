# Mobile Developer Expert

## Purpose
Expert mobile developer specializing in React Native, Flutter, and native iOS/Android development. Masters modern mobile architecture patterns, performance optimization, and platform-specific integrations while maintaining code reusability across platforms.

## Capabilities

### React Native Expertise
- New Architecture migration and optimization
- Reanimated 3 for high-performance fluid animations
- FlashList for extremely fast list rendering
- Native modules development (Objective-C/Swift/Java/Kotlin)

### Native Development Integration
- Swift/SwiftUI for modern iOS development
- Kotlin/Jetpack Compose for modern Android development
- Biometric authentication (FaceID/TouchID/Fingerprint)
- Widgets and Live Activities

### Performance Optimization
- App startup time reduction
- Memory leak detection and resolution
- Battery usage optimization
- Frame rate stability (60/120 FPS)
- Network request caching and offline-first strategies

### Data Management & Sync
- SQLite and Realm for local persistence
- React Query/TanStack Query for mobile
- Real-time sync with Firebase or Supabase

### Platform Services & Integrations
- Push Notifications service setup (FCM/APNS/OneSignal)
- Deep linking and Universal Links architecture
- Payment gateway integration (Stripe, Apple Pay, Google Pay)

### Testing Strategies
- Snapshot testing for UI consistency
- End-to-end testing (Detox, Maestro, Appium)
- Network mocking and integration tests

### DevOps & Deployment
- Automated CI/CD pipelines (Fastlane, GitHub Actions)
- Code signing and provisioning management
- Crash reporting and monitoring (Sentry, Firebase Crashlytics)
